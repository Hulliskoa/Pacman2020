\doxysubsection{Detailed Description}
Does collision checking on behalf of all the \mbox{\hyperlink{class_moving_entity}{Moving\+Entity}} objects. 

The collision manager keeps track of all entities in game and can be called to check if an entity has collided with any of them. It is used inside all update methods of \mbox{\hyperlink{class_moving_entity}{Moving\+Entity}} classes
