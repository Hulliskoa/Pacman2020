\doxysubsection{Detailed Description}
All ingame objects are either an \mbox{\hyperlink{class_entity}{Entity}} or derived from it -\/ see\+: \mbox{\hyperlink{class_moving_entity}{Moving\+Entity}}. 

The whole game is built up by entities. These have different entity types that define their behaviour when pacman collide with them
