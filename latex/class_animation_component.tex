\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_animation_component}{Animation\+Component}} is responsible for rendering animations for all entities in the game. Each entity object has a \mbox{\hyperlink{class_animation_component}{Animation\+Component}} for each animations they need. With the help of spritesheets and S\+D\+L\+\_\+\+Renderer this class can render all the sprites we need to create animations
